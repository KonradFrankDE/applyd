FAQ

Common questions.

Where is my data stored?
Exclusively in your browser (localStorage). No data leaves your device. No server, no cloud storage, no account.
What happens if I clear my browser cache?
Your data will be lost. Export regularly as a JSON backup — re-import takes one click.
Does Applyd work offline?
Yes, completely. No external dependencies, no CDN, no third-party fonts. The file runs entirely locally.
Can I use my data across multiple devices?
Not automatically — there's no sync. You can transfer data via JSON export and import it on another device.
Is the code open source?
Yes, MIT license. You can read, fork, modify, and distribute the code. The copyright notice must be retained.
How was Applyd built?
In collaboration with Claude (Anthropic) — without writing a single line of code manually. An experiment in vibe-coding and what's possible when you focus on the problem rather than the implementation.
